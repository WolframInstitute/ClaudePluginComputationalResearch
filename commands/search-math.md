Search external math resources using the `math-resources` skill.

Searches MathWorld, nLab, OEIS, DLMF, and Wikipedia math articles in
parallel, ranks the results, and optionally hands them to `resource-add`
to register in the wiki.

Pass keywords as arguments (e.g., `/search-math Bessel function asymptotics`).
For OEIS sequence lookup, pass the first few terms instead
(e.g., `/search-math 1 1 2 3 5 8 13`).
